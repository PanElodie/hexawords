Hexawords, c'est quoi ?

Le projet se définit par les 2 mots qui composent son intitulé : "Hexadecimal" et "Words".
Dans le monde du web, les couleurs sont caractérisées par un code de 6 (ou 3) caractères en base 16, d'où le mot "hexadécimal".

Composé de chiffres et de lettres (allant de A à F), il est parfois possible que le code corresponde à un mot (word en anglais). L'objectif de ce projet est donc d'illustrer un mot écrit en code hexadécimal tout en mettant en avant la couleur correspondante.

#D07

Comme vous avez pu le constater, le mot que j'ai choisi est DOT (#D07), le mot anglais pour "point".
Pour illustrer ce mot, j'ai placé différents points à des positions particulières afin qu'ils forment des mots et des figures qui s'animent.
Et ce n'est pas tout ! J'ai aussi agrémenté le tout avec des éléments sonores pour rendre ce site plus amusant et interactif.
J'espère que ce site vous aura plu !
